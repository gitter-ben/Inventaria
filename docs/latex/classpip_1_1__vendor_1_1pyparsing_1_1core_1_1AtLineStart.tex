\doxysubsection{Detailed Description}
\begin{DoxyVerb}Matches if an expression matches at the beginning of a line within
the parse string

Example::

    test = '''\
    AAA this line
    AAA and this line
      AAA but not this one
    B AAA and definitely not this one
    '''

    for t in (AtLineStart('AAA') + restOfLine).search_string(test):
        print(t)

prints::

    ['AAA', ' this line']
    ['AAA', ' and this line']\end{DoxyVerb}
