\begin{DoxyVerb}Matches if current position is at the beginning of a line within
the parse string

Example::

    test = '''\
    AAA this line
    AAA and this line
      AAA but not this one
    B AAA and definitely not this one
    '''

    for t in (LineStart() + 'AAA' + restOfLine).search_string(test):
        print(t)

    ['AAA', ' and this line']\end{DoxyVerb}
