\doxysubsection{Detailed Description}
\begin{DoxyVerb}A python-based backport of the `types.GenericAlias` class.

E.g. for ``t = list[int]``, ``t.__origin__`` is ``list`` and
``t.__args__`` is ``(int,)``.
